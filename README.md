# Student Knowledge Chatbot with Retrieval

## Project Overview
This project is a simple retrieval-based chatbot that answers user questions using information stored in a knowledge base.

## Features
- Interactive chatbot
- Retrieval-based question answering
- Multiple technology topics
- User-friendly interface

## Technologies Used
- Python

## Sample Questions
- What is Python?
- What is Java?
- What is SQL?
- What is Cyber Security?

## Output
The chatbot retrieves and displays the relevant answer from its knowledge base.
